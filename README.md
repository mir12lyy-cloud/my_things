# CasYYY的学习项目合集

## 介绍：
- 此仓库记载了本人在学习中制作的项目。绝大多数项目以C/C++为主，一些可能使用Python。

## 环境要求：
- C99以上（GCC）
- C++17以上（GCC，Clang，MSVC）
- Python 3.13以上

## 使用
1. **克隆仓库**
```bash
git clone https://github.com/mir12lyy-cloud/my_things.git
cd my_things
```
2. **阅读README** 
   项目内的README会介绍安装和使用的方法，以及内置函数，根据指引即可。
3. **环境要求**
    不同项目有具体的环境需求（比如C++标准），请根据项目内的README检查是否符合要求。
